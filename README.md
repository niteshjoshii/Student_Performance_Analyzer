# Student Performance Analyzer

A Streamlit-based web application that helps teachers analyze student academic performance using subject-wise marks, percentage calculation, result analysis, and interactive visualizations.

---

## Features

- Add student marks using a simple form
- Automatic total and percentage calculation
- Pass/Fail result generation
- Store student records in CSV format
- Interactive charts and analytics using Plotly
- Easy-to-use Streamlit interface

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly Express
